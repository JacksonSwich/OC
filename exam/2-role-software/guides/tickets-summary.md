## Чек-лист для подготовки

- [ ] Определи свой номер билета
- [ ] Посмотри список ПО для своего билета
- [ ] Для каждой программы подготовь ответ по схеме: **название → задачи → почему выбрал → альтернативы**
- [ ] Установи всё через `sudo bash installer.sh` (выбирая нужные категории)
- [ ] Выполни стартовую настройку (тема, шрифт, расширения)
- [ ] Настрой обмен данными (СУБД, Git, каталоги импорта/экспорта)
- [ ] Запусти `compatibility-setup.sh` (настройка совместимости)
- [ ] Проверь каждую программу: `--version`, запуск GUI, тестовый запрос
