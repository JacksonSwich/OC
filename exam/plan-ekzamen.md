# ЭТАП 1 — Установка ОС и защита (24.50 балла)

## Шаг 1.1 — Настройка ядра [2.50 балла]

```bash
sudo -i
cp /etc/sysctl.conf /etc/sysctl.conf.backup
cat >> /etc/sysctl.conf << EOF

net.ipv4.tcp_syncookies = 1
net.ipv4.conf.all.rp_filter = 1
vm.swappiness = 10
net.core.somaxconn = 65535
fs.file-max = 100000
kernel.pid_max = 65536
EOF
sysctl -p
```

**Проверка:** `sysctl vm.swappiness` → должен показать 10  
**Преподу сказать:** "Увеличил производительность — уменьшил swappiness до 10, увеличил лимит соединений, включил syn-cookies от DDoS."

## Шаг 1.2 — SSH + Удаленный доступ [4.00 балла]

```bash
# SSH (2.0 балла)
apt update
apt install -y openssh-server
systemctl enable ssh --now
systemctl status ssh --no-pager  # проверить
ss -tlnp | grep :22              # показать порт
```

**Проверка:** `systemctl status ssh` → running  
**Преподу сказать:** "SSH — защищённый протокол для удалённого управления. Порт 22 слушается."

```bash
# XRDP на всякий случай (препод может попросить графику)
apt install -y xrdp
systemctl enable xrdp --now
systemctl status xrdp --no-pager
```

**Преподу сказать:** "SSH — основной, xrdp — запасной вариант для графического доступа через RDP."

## Шаг 1.9 — Группы и права [2.00 балла]

```bash
groupadd developers
groupadd admins
usermod -aG developers $USER
usermod -aG admins $USER
useradd -m -G developers -s /bin/bash testuser
echo "testuser:testpass123" | chpasswd

mkdir -p /srv/projects
chown root:developers /srv/projects
chmod 775 /srv/projects

mkdir -p /srv/admin
chown root:admins /srv/admin
chmod 770 /srv/admin

ls -la /srv/
```

**Преподу сказать:** "Groups: developers и admins. 775 на projects — читают/пишут, 770 на admin — только админы."

## Шаг 1.10 — Аутентификация + Мониторинг [2.00 балла]

```bash
# Политика паролей + chage
apt install -y libpam-pwquality
sed -i 's/# minlen = 9/minlen = 8/' /etc/security/pwquality.conf
chage -M 90 $USER     # пароль на 90 дней
chage -W 7 $USER      # предупреждение за 7 дней

# Аудит
apt install -y auditd audispd-plugins logwatch
systemctl enable auditd --now
auditctl -e 1
auditctl -w /etc/passwd -p wa -k passwd_changes
auditctl -l  # проверить
```

**Преподу сказать:** "Пароль минимум 8 символов, смена каждые 90 дней. Auditd следит за изменениями /etc/passwd. Logwatch анализирует логи системы."
